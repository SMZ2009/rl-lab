# Chapter 2 蒙特卡洛 MC 与时序差分 TD 算法

上回我们聊完了 MDP 和贝尔曼方程，但是转移概率 P 和奖励函数 R 往往都是环境内部的秘密，我们根本拿不到。

没有普适公式？那就从真实数据中入手。通过**蒙特卡洛 MC**和**时序差分 TD**两大家族的无模型算法，不依赖 P 和 R 的显式表达式，只靠智能体跟环境“交手”产生的经验来估计价值函数，进而找到好策略。

# 1. 蒙特卡洛方法 MC

把一整条轨迹跑到结束，然后统计实际获得的回报，用这个回报来更新价值估计。

因为我们在用随机采样的方式去估计期望，理论上要对所有可能的轨迹求期望，但轨迹数量是指数级的，算不动。MC 的做法是：跑 N 局游戏，每局都从 s 出发，记录每局的实际回报 G，然后取平均。大数定律告诉我们，样本越多，平均值越接近真实期望。

MC 方法只适用于有终止状态的回合制任务（episodic tasks），不能用于持续运行的任务（比如终身学习机器人），因为那样永远等不到“结束”来算总回报。


- 优点：无偏估计，并且不需要知道 P 和 R。

- 缺点：必须等整局结束才能更新，不能实时学习；方差大（因为一条轨迹的随机性可能很大）；只适合回合制任务。

# 2. 时序差分方法 TD

MC 用实际回报 $G_t$ 作为目标来更新 $V(s_t)$，但是 $G_t$ 是未来的总和，必须等终局。TD 则用当前奖励 + 下一步估计价值来作为目标，即：

$$
V(S_t) \leftarrow V(S_t) + \alpha \bigl[ R_{t+1} + \gamma V(S_{t+1}) - V(S_t) \bigr]
$$

实际上就是利用 $R_{t+1} + \gamma V(S_{t+1})$ 和 $V(S_t)$ 的残差（TD误差）进行更新。

$$
\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)
$$

举个例子：从A地到C地，一开始预计花费1000分钟。但是到了途中B地之后，实际A到B花了300分钟，再预计B到C只需要600分钟，也就是总时长只需900分钟。

在这里，TD误差就是300+600-1000为100分钟。我不需要抵达C地，就能通过在中间点进行预测来更新价值函数了。

所以，TD相比MC来说，可以不等整局结束就学习，所以在持续运行的任务上也可以使用。

# 3. TD 估计动作价值：SARSA 和 Q-learning

## 3.1 SARSA

是一种 **在线 on-policy TD 控制**。

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \bigl[ R_{t+1} + \gamma Q(S_{t+1}, A_{t+1}) - Q(S_t, A_t) \bigr]$$

名字来自五元组 \((S_t, A_t, R_{t+1}, S_{t+1}, A_{t+1})\)。它**使用当前策略 $\pi$ 产生的下一步动作** \(A_{t+1}\) 来构造目标，所以它评估和改进的是同一个策略（on-policy）。

## 3.2 Q-learning

是一种 **离线 off-policy TD 控制**。

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \bigl[ R_{t+1} + \gamma \max_{a'} Q(S_{t+1}, a') - Q(S_t, A_t) \bigr]$$

它**直接用最大化操作**，不依赖下一步实际采样的动作。

所以它可以一边用行为策略（比如 \(\varepsilon\)-贪心）探索，一边学习最优策略（off-policy）。

Q-learning 是 DQN 的前身，以后会大放异彩。

---

## 3.3 怎么理解在线与离线？

关键在于区分 Agent 在探索环境时使用的策略（action policy）和在学习和优化时使用的策略（目标策略）是否一致。

SARSA 边探索边学习当前策略的价值，Q-learning 可以用任意策略进行探索，同时学习最优策略。

显然，从感知上，离线学习会收敛更快。但是在对安全和稳定性要求高的场景中，在线学习更有优势。
