# Chapter 3 DQN

上一章我们聊了无模型价值学习的两大支柱：MC 和 TD。TD 的佼佼者 Q-learning 更是直接给 DQN 铺好了路。

但 Q-learning 有个“富贵病”：它得把每个状态-动作对的价值 Q(s,a) 都记在一张表格里。状态空间稍微大一点，表格就爆炸了。

这样太占空间了！那怎么办？我们可以通过训练一个神经网络来拟合价值函数，也就是 Deep Q Network（DQN）。

但 DQN 只关心“平均回报”，而现实中的风险、随机性都被平均掉了——于是后来者又提出了分布强化学习，其中隐式分位数网络（IQN）更是把回报的整个分布都学会了，让智能体既能看清“平均收益”，也能掂量“波动风险”。

# 1. 深度 Q 网络 DQN

Q-learning 的更新式：

$$Q(s,a)\leftarrow Q(s,a)+\alpha\bigl[r+\gamma\max_{a'}Q(s',a')-Q(s,a)\bigr]$$

我们用神经网络 $Q(s,a;\theta)$ 来拟合所有状态-动作的价值，参数 $\theta$ 就是网络的权重。输入状态 s，输出每个动作的 Q 值。

训练时，我们最小化 TD 误差的平方：

$$\mathcal{L}(\theta)=\mathbb{E}_{(s,a,r,s')}\left[\left(r+\gamma\max_{a'}Q_{\text{target}}(s',a';\theta^-)-Q(s,a;\theta)\right)^2\right]$$

---

为什么不能用神经网络直接套 Q-learning？

- **SARSA 连续的几步高度相关**，非常容易过拟合。

- **目标不固定**，TD 目标里也有 Q，像“一个人追着自己的影子跑”，参数剧烈波动。

---

怎么解决？

- **经验回放（Experience Replay）**

把智能体每次交互得到的四元组 $(s,a,r,s')$ 存进一个“记忆池”里。训练时，从记忆池中随机抽取一小批（mini-batch）样本来更新网络。这样打破了时间上的相关性，而且样本可以重复利用，数据效率更高。

- **目标网络（Target Network）**

我们复制一个结构相同的网络 \(Q_{\text{target}}\)，它的参数 \(\theta^-\) 每隔 \(C\) 步才从当前网络 \(\theta\) 拷贝一次。计算 TD 目标时，用 \(Q_{\text{target}}\) 的输出来构造：

$$y = r + \gamma \max_{a'} Q_{\text{target}}(s', a'; \theta^-)$$

这样目标在 C 步内是固定的，学习过程稳定多了。

---

- 优点：首次让深度强化学习在图像输入下直接学到好策略，开创性强；经验回放和目标网络成为后续算法的标配。

- 缺点：只能处理离散动作空间；对超参数敏感；仍只估计期望 Q 值，丢失了回报分布的方差、偏度等信息（比如某些动作平均回报高但风险极大，DQN 看不出来）。
