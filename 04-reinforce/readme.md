# Chapter 4 策略梯度定理与 REINFORCE 算法

在 Chapter 1 - Chapter 3 中，我们学习了一系列依靠价值函数进行学习的方法。

从最基本的基于马尔可夫决策过程数学模型 MDP，再到将价值函数写成递推式（贝尔曼方程），使其可以被训练。

然后我们学习了最基本的两种思想：蒙特卡洛 MC 以及时序差分 TD。蒙特卡洛算法需要一个结局，根据总的 $G_t$ 实际回报来更新学习，但是时序差分方法就不需要，可以允许每一步都来进行一个学习，并能够得到相同的更新效果。

基于 TD，我们又学习了两种不同策略：在线策略 SARSA，以及离线策略 Q-learning。我们了解到在线策略非常稳定，因为它依赖既定的策略去执行动作；离线策略不稳定但是收敛快，并且能够探索更多的动作路径。

然后我们介绍了 DQN，将 Q-learning 需要占用极大内存空间的算法通过神经网络来近似。并且了解到 DQN 是如何通过 **经验回放** 和 **目标网络** 来使训练的稳定性增强。

从 Chapter 4 开始，我们开始接触另外一套强化学习路径，也就是直接学习 **策略函数 $\pi$**。

# 1. 策略参数化与优化目标

在价值学习中，我们优化的对象是 Q 值或 V 值；而在策略梯度中，我们直接优化策略的参数 \(\theta\)。

策略可以是一个神经网络：输入状态 \(s\)，输出动作的概率分布（离散动作）或高斯分布的均值和方差（连续动作）。

优化目标是**期望累积回报**：

\[
J(\theta) = \mathbb{E}_{\tau \sim \pi_\theta} \left[ G(\tau) \right]
\]

其中 \(\tau = (s_0, a_0, r_1, s_1, a_1, \dots, s_T)\) 是由策略 \(\pi_\theta\) 采样得到的一条轨迹，\(G(\tau) = \sum_{t=0}^{T-1} \gamma^t r_{t+1}\) 是总折扣回报。

我们的目标是找到 \(\theta^*\) 使得 \(J(\theta)\) 最大。直接用有限差分去估计梯度既低效又高方差，因此我们需要一个优雅的数学工具——**策略梯度定理**。

# 2. 策略梯度定理

策略梯度定理告诉我们：目标函数的梯度可以写成**期望形式**，且不需要知道环境动力学（转移概率）的梯度。这对强化学习极其重要，因为我们通常无法对环境求导。

对于任意可微的策略 \(\pi_\theta(a|s)\)，有：

\[
\nabla_\theta J(\theta) = \mathbb{E}_{\tau \sim \pi_\theta} \left[ \sum_{t=0}^{T-1} \nabla_\theta \log \pi_\theta(a_t|s_t) \, G_t \right]
\]

其中 \(G_t = \sum_{k=t}^{T-1} \gamma^{k-t} r_{k+1}\) 是从时刻 \(t\) 开始的折扣回报。

**证明思路**（简略）：

利用似然比技巧 \(\nabla_\theta \pi_\theta = \pi_\theta \nabla_\theta \log \pi_\theta\)，将梯度从环境中“剥离”出来。具体地，轨迹的概率为：

\[
p_\theta(\tau) = p(s_0) \prod_{t=0}^{T-1} \pi_\theta(a_t|s_t) \, p(s_{t+1}|s_t, a_t)
\]

对 \(J(\theta) = \int p_\theta(\tau) G(\tau) d\tau\) 求导，只有策略项依赖 \(\theta\)，转移概率的导数项全部消失，最终得到上述期望形式。

**关键洞察**：\(\nabla_\theta \log \pi_\theta(a|s)\) 称为**得分函数**，它指示了参数变化如何影响动作的概率。梯度方向就是：**增加那些带来高回报的动作的概率，降低带来低回报的动作的概率**。

# 3. REINFORCE 算法

有了策略梯度定理，我们可以直接设计蒙特卡洛算法——**REINFORCE**（Williams, 1992）。

它是最原始、最简单的策略梯度方法。

算法流程如下：

1. 用当前策略 \(\pi_\theta\) 采样一条完整轨迹 \((s_0, a_0, r_1, \dots, s_{T-1}, a_{T-1}, r_T)\)。
2. 对每个时刻 \(t\)，计算从 \(t\) 开始的折扣回报 \(G_t = \sum_{k=t}^{T-1} \gamma^{k-t} r_{k+1}\)。
3. 更新参数：

\[
\theta \leftarrow \theta + \alpha \, \gamma^t \, G_t \, \nabla_\theta \log \pi_\theta(a_t|s_t)
\]

（有些版本直接使用 \(G_t\) 而不乘 \(\gamma^t\)，这里按原始推导的梯度形式，乘上 \(\gamma^t\) 实际上对应的是折扣因子在梯度中的位置，不影响本质——实际实现中常用 \(G_t\) 本身已包含折扣，所以简化为 \(\alpha G_t \nabla_\theta \log \pi_\theta(a_t|s_t)\)。）

4. 重复上述过程直至收敛。

**优点**：
- **理论简洁**：直接源自策略梯度定理，实现起来非常直观。
- **适用连续动作**：只需将策略网络输出高斯分布参数，采样动作即可，无需处理离散动作的 \(\max\)。
- **无偏估计**：因为 \(G_t\) 是真实回报的无偏采样，梯度估计在期望上无偏（尽管方差大）。

**缺点**：
- **高方差**：\(G_t\) 的波动非常大（包含从 \(t\) 到终点的所有随机性（策略随机、转移随机）），导致梯度估计噪声严重，学习慢且不稳定。
- **必须完整轨迹**：无法在线更新，必须等一个 episode 结束，不适合情节很长或没有终止环境的情况。
- **收敛慢**：由于高方差，需要很小的学习率，训练效率低。

---

怎么缓解高方差？引入**基线（Baseline）**。

我们可以从梯度中减去一个只与状态有关的基线 \(b(s_t)\)，而不改变期望：

\[
\nabla_\theta J(\theta) = \mathbb{E} \left[ \sum_t \nabla_\theta \log \pi_\theta(a_t|s_t) \, (G_t - b(s_t)) \right]
\]

因为 \(\mathbb{E}_{a_t \sim \pi_\theta} [\nabla_\theta \log \pi_\theta(a_t|s_t) \, b(s_t)] = 0\)，所以期望不变，但方差可以大幅降低。常用的基线是状态价值函数 \(V(s_t)\)，于是 \(G_t - V(s_t)\) 就是**优势函数**\(A(t)\)的蒙特卡洛估计。

\( V \) 把“环境本身的随机性（状态好坏）”给剔除掉了，只保留“动作带来的独特影响”。
