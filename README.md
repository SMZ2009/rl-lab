## 前情提要

接下来，我将开启我的强化学习算法分享之路。

老实说，现在世面上的强化学习算法并不是很“小白友好”，能够零基础入门。所以作为一名在Lab里面一直调整Robot的Policy，并且被+1轰炸PPO、GRPO、SAO各种新奇名词的准大二学生，我想要自己动手丰衣足食。

所以，我将用最直白、最通俗易懂、最不绕弯子的方法来介绍强化学习。
